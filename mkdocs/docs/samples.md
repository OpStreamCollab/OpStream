# Live samples gallery

Every sample below runs **live** on the public OpStream demo server
([hostdemo.opstream.stream](https://hostdemo.opstream.stream/samples)) and links to
its source in this repo. Open any demo in two browser tabs to see real-time
collaboration — they all talk to the same OpStream server.

!!! tip "Zero setup"
    No install needed to try them. To run a sample locally instead, follow the
    README in its source folder (`docker run -p 8080:8080 opstreamcollab/opstream`,
    then `npm install && npm run dev` for the JS samples).
